Column Grid

Tradition 12 column grid based on display: inline-block; instead of floats and margins. Grid container .grid-columns sets the font size to 0 and resets it to $font-size in its child .column elements. This means elements immediately after .grid-columns must contain the class .column, or they will not display its contents. Columns are percentage based, so this grid adjusts to the width of its container, or whatever you set it to.
